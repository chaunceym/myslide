文档编辑

如果你会 Markdown 语法，那么这个软件对你来说上手即会，如果你还是个新手,那么也没关系，我来教你一些简单的 Markdown 语法

  • # 首页
  • ## 除首页的其他的一级页面
  • ### 二级页面
  • ##### 一级标题
  • ###### 二级标题
  • -/+/* 无序列表
  • 1. 有序列表
  • **字** 加粗字体
  • *字* 倾斜字体
  • ***字*** 加粗倾斜字体
  • [超链接名](超链接地址 "超链接title") 引入超链接

当你看完这些就能运用 Markdown 的基本功能，如果还想学更多的语法，请看 Markdown 中文文档
